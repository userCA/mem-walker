**模板类型**: 通用AI系统/Agent架构评审
**适用场景**: 新系统设计评审、现有系统重构评审、第三方系统技术评估、AI记忆服务专项评审
**版本**: 2.0

**基于**: v1.0 + mnemosyne项目实践 + AI系统工程最佳实践

## 使用说明

### 适用对象
- AI推理服务（LLM、RAG、Agent、多模态）
- 机器学习平台（特征工程、模型训练、模型服务）
- 数据管道系统（ETL、流处理、实时分析）
- 微服务架构应用
- 混合AI系统（AI + 传统微服务）
- Agent记忆系统（如RAG、向量数据库、知识图谱）
- 多Agent协作系统

### 评审前置条件
在执行评审前，需收集以下信息：
1. 系统架构设计文档（ASCII图或Mermaid图）
2. 核心技术栈清单（含版本）
3. 业务场景与用户故事
4. 非功能性需求文档（SLA、容量规划）
5. 已知约束条件（预算、合规、技术债务）
6. AI模型清单（版本、供应商、配额）
7. Prompt模板库及版本记录

## 附录：评审辅助工具

### A. 架构图检查清单
- [ ] 是否使用标准符号（ UML / C4 / Mermaid）
- [ ] 是否标注数据流向
- [ ] 是否标注信任边界
- [ ] 是否标注部署视图
- [ ] 是否标注扩展点
- [ ] 是否标注AI组件（LLM/Embedding/VectorStore）
- [ ] 是否标注记忆流（召回/存储/遗忘）

### B. 代码质量扫描命令
```bash
# 静态分析
ruff check .

# 类型检查
mypy .

# 测试覆盖
pytest --cov=src --cov-report=term-missing

# 依赖审计
pip-audit

# Promptlint（如果使用Prompt模板）
promptlint .
```

### C. 安全扫描命令
```bash
# 漏洞扫描
trivy fs .

# 依赖安全
snyk test

# Secrets扫描
gitleaks detect

# AI安全扫描
# 检查Prompt注入风险
python -m safety check
```

### D. RAG质量评估命令
```bash
# RAGAS评估（如已集成）
python -m ragas evaluate --dataset=test_set.json

# 召回率测试
python -m mnemosyne.benchmark --recall-test

# 延迟基准测试
python -m mnemosyne.benchmark --latency-test
```

### E. 成本估算参考
```bash
# Token消耗估算
python -m mnemosyne estimate --query-file=queries.json

# 月度成本预估
python -m mnemosyne cost-estimate --monthly-volume=10000
```
